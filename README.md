## 🚀 Featured Projects

### 🧬 [Noema — AI Second Brain](https://github.com/chaitanyapawar-dev/noema)
> Local-first multimodal AI knowledge engine. Zero cloud cost, 100% yours.

- 7-stage async ingestion pipeline — Whisper ASR + OCR + BLIP captions + CLIP embeddings
- Hybrid text + visual semantic search, reranked by local LLM (Ollama/Llama 3.2)
- Conversational RAG chat with source citations
- **Stack:** Next.js 14 · FastAPI · ChromaDB · PyTorch · sentence-transformers

---

### 🛰️ [Satellite Drought Risk Prediction](https://github.com/chaitanyapawar-dev)
> Custom CNN that reads satellite images and tells you how worried to be about drought.

- 3-block CNN trained from scratch on 27,000 EuroSAT images · **95% test accuracy · 0.998 AUC**
- Maps 10 land-cover classes to contextual drought risk levels
- Live Streamlit app for real-time classification + advisory generation
- **Stack:** Python · TensorFlow · Streamlit · EuroSAT (Sentinel-2)

---

### 🎙️ [Virtual Assistant](https://github.com/chaitanyapawar-dev)
> Wake-word activated AI assistant. Says your name, wakes up, handles 12 intent categories.

- Custom wake-word activation · Google Gemini API for structured intent parsing
- Hindi TTS replies · JWT auth · persistent command history in MongoDB
- **Stack:** React 19 · Node.js · Express · MongoDB · Gemini API
